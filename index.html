为 Mac 用户打造的语音输入

不是云端转写服务，而是驻留在菜单栏的本地助手——快、稳、私密。

完全离线，本地 ASR

内置 Qwen3-ASR 模型，在 Mac 上直接完成语音识别。没有 API 调用， 没有按量计费，断网也能用。

  • 首次下载模型后永久本地运行

数据不出设备

语音在本地处理，不经过任何第三方服务器。适合处理敏感文档、代码与客户信息。

  • 兼容绝大多数 macOS 应用
